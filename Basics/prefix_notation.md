

# prefixes

- member varibles
    - m_ : member variable

- Pointers
    - p : shared_ptr or ComPtr
	- up : unique_ptr
	- wp : weak_ptr
	- rp : raw pointer

- Integers
    - dw : DWORD
	- ui : UINT
    	- ui8, ui64 equals UINT8, UINT64

- floating point
    - f : float
	- d : double

- linear algebras
    - v : mostly Vector3(in Simplemath)
	- mat : Matrix(in Simplemath, 4x4 Matrix)
